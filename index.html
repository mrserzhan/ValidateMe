ValidateMe

Powerfull jQuery validation module

Using

Add this code to <head> section of your html code

html code 
<script type="text/javascript" src="js/jquery.min.js"></script>
<script type="text/javascript" src="js/jquery.validate.js"></script>

For validating some object add code like this

javasript code 
$(document).ready(function(){
    $("#id1").validate();
});

Example 1. Adding Rules

Lets add some rules for out input. ValidateMe has default functions for validating. All functions you can watch later. Okay. For example, I use rule "reqired".


javascript code 
$("#id1").validate({
    rules: {
        'required' : {
            'msg': 'This input required'
        }
    },
    'default': 'All right'
});

Example 2. Passing options

Some functions use options. For example length function can use options as like 'min', 'max'. Perhaps the meaning of this two you have learned.


javascript code 
$("#id2").validate({
    rules: {
        'length' : {
            'msg': 'Length of this must be longer or equal to 2, less or equal to 6, 
must not equal to 3 and can be only 4,5 or 6',
            'min': 2,
            'max': 6,
            'not_equal': 3,
            'equals': [4,5,6]
        }
    },
    'default': 'All right'
});

Example 3. Using an another place for showing status

ValidateMe support using another place for show status. It possible with 'displayTo' option. You must pass there a jQuery object for showin status on it.


javascript code 
$("#id3").validate({
    rules: {
        'required' : {
            'msg': 'This input required'
        }
    },
    'displayTo' : $("#another_place"),
    'default': 'All right'
});

Example 4. Using another functions.

You can use your functions for validating. For it you just write your function name with prepending "function_" string


javascript code 
function prefix_must_salem(str, options)
{
    return str.substr(0,5) == "salem";
}

$(document).ready(function(){

    $("#id4").validate({
        rules: {
            'function_prefix_must_salem' : {
                'msg': 'Prefix must be "salem"'
            }
        },
        'default': 'All right'
    });

});
